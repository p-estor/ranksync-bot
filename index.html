RankSync Bot - Sincronización de Rangos de LoL/TFT para Discord

RankSync es un bot de Discord que permite a los usuarios vincular sus cuentas de League of Legends y Teamfight Tactics para obtener automáticamente roles de rango actualizados en su servidor de Discord.

Demostración del Flujo de Usuario (User Flow)

Paso 1: Vincular una Cuenta de Riot Games (`/vincular`)

Los usuarios inician el proceso de vinculación con el comando /vincular, introducen su Riot ID y verifican la propiedad de la cuenta cambiando su icono de invocador. Una vez verificado, sus rangos son consultados y los roles de Discord se asignan automáticamente.

Paso 2: Actualizar Rangos y Roles (`/refresh`)

Los usuarios pueden actualizar sus rangos y roles en cualquier momento para reflejar los cambios en sus partidas recientes.

Paso 3: Ver y Desvincular Cuentas (`/viewaccounts`)

Los usuarios pueden ver todas las cuentas de Riot vinculadas a su Discord y tienen la opción de desvincular cualquiera de ellas.

Información Técnica

Desarrollado con Node.js y Discord.js, utilizando la API de Riot Games (via TeemoJS) y SQLite para la gestión de datos locales.

Funcionalidades clave: Vinculación segura de cuentas, verificación de propiedad, actualización dinámica de rangos y roles de Discord, soporte para múltiples cuentas por usuario.

Compromiso con las Políticas de Riot Games

El bot RankSync cumple plenamente con los Términos de Servicio de Riot Games y las Políticas de Desarrollo. No almacena datos sensibles del usuario. Todas las operaciones de la API de Riot se realizan de acuerdo con las directrices establecidas.
